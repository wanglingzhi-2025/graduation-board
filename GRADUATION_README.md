# 🎓 毕业留言板系统

一个专为初中生设计的毕业留言板系统，让同学和老师们记录美好时光，珍藏青春回忆。

## ✨ 功能特点

### 核心功能
- 🔐 **用户认证系统**
  - 45个预设学生账号 (stu001-stu045)
  - 10个预设教师账号 (tch001-tch010)
  - 默认密码与用户名相同
  
- 🏫 **学校班级选择**
  - 支持多个学校选择
  - 默认：第一中学 8年2班
  - 可选择不同班级

- 💬 **留言功能**
  - 公开留言：所有人可见
  - 私密留言：仅接收者可见
  - 支持留言标题和内容
  - 字数限制：500字

- 📷 **照片上传**
  - 支持上传照片到留言
  - 实时预览功能
  - 可删除已上传照片

- ❤️ **互动功能**
  - 点赞功能
  - 查看留言详情
  - 统计我的留言和收到的留言

- 🔍 **筛选和搜索**
  - 全部留言
  - 公开留言
  - 私密留言
  - 我的留言
  - 关键词搜索

## 🚀 快速开始

### 1. 打开登录页面
在浏览器中打开 `graduation.html`

### 2. 选择学校和班级
- 默认选择：第一中学 8年2班
- 可以选择其他学校和班级

### 3. 登录账号

**学生账号示例：**
- 用户名：stu001, stu002, ... stu045
- 密码：与用户名相同

**教师账号示例：**
- 用户名：tch001, tch002, ... tch010
- 密码：与用户名相同

### 4. 开始使用
登录后即可：
- 查看所有留言
- 写新留言
- 上传照片
- 点赞互动

## 📝 使用说明

### 写留言
1. 点击"✍️ 写留言"按钮
2. 选择留言类型（公开/私密）
3. 如果是私密留言，选择接收者
4. 填写标题和内容
5. 可选：上传照片
6. 点击"发布留言"

### 查看留言
- 点击任意留言卡片查看详情
- 可以点赞喜欢的留言
- 使用筛选按钮查看不同类型的留言

### 搜索留言
- 在搜索框输入关键词
- 可搜索标题、内容或作者名称

## 💾 数据存储

系统使用浏览器的 localStorage 存储数据：
- `graduation_accounts`: 用户账号信息
- `graduation_current_user`: 当前登录用户
- `graduation_messages`: 所有留言数据
- `graduation_accounts_initialized`: 初始化标记

## 🔒 隐私保护

- 私密留言仅发送者和接收者可见
- 每个用户只能看到与自己相关的私密留言
- 公开留言对所有人可见

## 💡 使用提示

1. **首次使用**：系统会自动初始化所有账号
2. **密码管理**：默认密码与用户名相同，建议实际使用时修改
3. **照片大小**：建议上传小于5MB的照片以获得最佳性能
4. **数据备份**：重要留言建议截图保存
5. **浏览器缓存**：清除浏览器缓存会删除所有数据

## 📞 常见问题

**Q: 忘记密码怎么办？**
A: 默认密码与用户名相同。如果修改后忘记，需要清除浏览器数据重新初始化。

**Q: 可以修改个人信息吗？**
A: 当前版本使用预设账号，名称为"学生X"或"教师X"。

**Q: 留言可以删除吗？**
A: 当前版本不支持删除，请谨慎发布。

**Q: 数据会丢失吗？**
A: 数据存储在浏览器本地，清除浏览器数据会导致丢失。

**Q: 可以导出留言吗？**
A: 建议使用截图或浏览器打印功能保存重要留言。
