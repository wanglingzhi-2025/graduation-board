# 📚 背单词应用 (Vocabulary Learning App)

一个功能完整的网页版背单词应用，支持用户注册登录、单词管理、每日打卡和测验功能。

## ✨ 功能特点

### 🔐 用户认证
- 用户注册与登录
- 密码加密存储
- 会话管理
- 多用户支持

### 📅 每日登录追踪
- 自动记录登录日期
- 连续登录天数统计
- 登录打卡激励

### 📖 单词管理
- 添加新单词（单词、释义、例句、分类）
- 编辑已有单词
- 删除单词
- 搜索单词（支持单词和释义搜索）
- 按分类筛选
- 多种排序方式（时间、字母、复习次数）
- 单词统计信息

### 🎮 测验系统
- 多选题测验（显示单词，选择正确释义）
- 可配置题目数量（5/10/15/20题）
- 按分类筛选测验单词
- 实时计时
- 答题进度显示
- 即时反馈（正确/错误）
- 测验结果统计
- 错题回顾
- 测验历史记录

### 📊 数据统计
- 单词总数
- 连续登录天数
- 测验次数
- 平均分数
- 最近测验记录
- 单词复习统计

## 🚀 快速开始

### 安装

1. 下载或克隆项目到本地
2. 无需安装任何依赖，直接在浏览器中打开即可

### 使用方法

1. 在浏览器中打开 `index.html`
2. 注册新账号或使用已有账号登录
3. 开始添加单词并进行学习

## 💾 数据存储

应用使用浏览器的 localStorage 存储所有数据，包括：

- **用户信息**：用户名、密码（加密）、创建时间、登录信息
- **单词数据**：单词、释义、例句、分类、添加时间、复习统计
- **测验记录**：测验时间、分数、题目数量、用时

### 数据结构

#### 用户对象
```javascript
{
  username: string,
  password: string (hashed),
  createdAt: timestamp,
  lastLogin: timestamp,
  loginStreak: number
}
```

#### 单词对象
```javascript
{
  id: string,
  word: string,
  definition: string,
  example: string,
  category: string,
  addedDate: timestamp,
  reviewCount: number,
  correctCount: number,
  incorrectCount: number
}
```

#### 测验对象
```javascript
{
  id: string,
  date: timestamp,
  score: number,
  totalQuestions: number,
  duration: number,
  words: array
}
```

## 🎯 使用指南

### 注册与登录

1. 首次使用需要注册账号
2. 用户名：3-20个字符，支持字母、数字、下划线和中文
3. 密码：至少6个字符
4. 注册成功后自动跳转到登录页面

### 添加单词

1. 进入"单词本"页面
2. 点击"➕ 添加单词"按钮
3. 填写单词信息：
   - 单词（必填）
   - 释义（必填）
   - 例句（可选）
   - 分类（可选，支持自动补全）
4. 点击"保存"

### 管理单词

- **搜索**：在搜索框输入关键词，支持搜索单词和释义
- **筛选**：选择分类进行筛选
- **排序**：支持按时间、字母、复习次数排序
- **编辑**：点击单词卡片上的"编辑"按钮
- **删除**：点击"删除"按钮，确认后删除

### 开始测验

1. 进入"开始测验"页面
2. 选择题目数量（5/10/15/20题）
3. 选择分类（可选，默认所有分类）
4. 点击"开始测验"
5. 依次回答问题
6. 查看测验结果和错题回顾

### 查看统计

在主页可以看到：
- 连续登录天数
- 单词总数
- 测验次数
- 平均分数
- 最近测验记录

## 📝 使用技巧

1. **每天坚持登录**：保持登录连续性，养成学习习惯
2. **合理分类**：为单词添加分类，便于系统学习
3. **定期测验**：通过测验检验学习效果
4. **复习错题**：重点关注测验中的错题
5. **添加例句**：例句有助于理解和记忆单词
6. **循序渐进**：从少量单词开始，逐步增加

## 🔧 技术栈

- **前端**：HTML5, CSS3, Vanilla JavaScript
- **存储**：localStorage
- **架构**：模块化设计，关注点分离

## 📊 性能优化

- 防抖搜索，减少不必要的渲染
- 懒加载，按需加载数据
- 缓存机制，提高响应速度
- 最小化 DOM 操作

## 🐛 已知限制

- 数据存储在浏览器本地，清除浏览器数据会丢失
- 不支持跨设备同步
- localStorage 有存储容量限制（通常 5-10MB）
- 单用户模式（同一浏览器同一时间只能登录一个账号）
