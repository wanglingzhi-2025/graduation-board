# 📱 毕业留言板 - 手机发布指南

## 🎯 发布方式概览

将毕业留言板发布到手机上有以下几种方式：

### 方式一：使用免费托管平台（推荐）⭐
**优点**：最简单、免费、可多人同时访问
**适合**：想要分享给全班同学使用

### 方式二：本地网络共享
**优点**：数据完全私密、不需要上传
**适合**：仅在同一WiFi环境下使用

### 方式三：使用云存储服务
**优点**：简单快捷、可离线访问
**适合**：个人使用或小范围分享

---

## 🌟 方式一：使用免费托管平台（推荐）

### 选项 A：GitHub Pages（最推荐）

#### 步骤 1：准备 GitHub 账号
1. 访问 https://github.com
2. 注册一个免费账号（如果还没有）

#### 步骤 2：创建仓库
1. 登录 GitHub
2. 点击右上角的 "+" → "New repository"
3. 仓库名称填写：`graduation-board`
4. 选择 "Public"（公开）
5. 勾选 "Add a README file"
6. 点击 "Create repository"

#### 步骤 3：上传文件
1. 在仓库页面，点击 "Add file" → "Upload files"
2. 将以下文件拖拽上传：
   ```
   graduation.html
   message-board.html
   css/graduation.css
   js/graduation-auth.js
   js/graduation-board.js
   ```
3. 在底部填写提交信息：`上传毕业留言板文件`
4. 点击 "Commit changes"

#### 步骤 4：启用 GitHub Pages
1. 在仓库页面，点击 "Settings"（设置）
2. 在左侧菜单找到 "Pages"
3. 在 "Source" 下选择 "main" 分支
4. 点击 "Save"
5. 等待几分钟，页面会显示访问链接

#### 步骤 5：访问网站
- 你的网站地址将是：`https://你的用户名.github.io/graduation-board/graduation.html`
- 用手机浏览器打开这个链接即可使用
- 分享这个链接给同学们

**📱 手机访问方式：**
- 直接在手机浏览器输入网址
- 或生成二维码让同学扫描（可使用在线二维码生成器）
- 建议添加到手机主屏幕，像APP一样使用

---

### 选项 B：Netlify（操作更简单）

#### 步骤 1：准备文件
1. 将所有文件放在一个文件夹中
2. 将文件夹压缩成 ZIP 文件

#### 步骤 2：上传到 Netlify
1. 访问 https://www.netlify.com
2. 注册免费账号
3. 点击 "Add new site" → "Deploy manually"
4. 将 ZIP 文件拖拽到页面上
5. 等待部署完成

#### 步骤 3：获取链接
- Netlify 会自动生成一个网址，如：`https://random-name-123.netlify.app`
- 可以在设置中自定义域名前缀
- 用手机浏览器访问这个链接

---

### 选项 C：Vercel（速度快）

#### 步骤 1：注册账号
1. 访问 https://vercel.com
2. 使用 GitHub 账号登录

#### 步骤 2：导入项目
1. 点击 "Add New" → "Project"
2. 选择 "Import Git Repository"
3. 连接你的 GitHub 仓库
4. 点击 "Deploy"

#### 步骤 3：访问网站
- 部署完成后会得到一个网址
- 用手机访问即可

---

## 🏠 方式二：本地网络共享

### 使用 Python 服务器（你已经在用）

#### 当前状态
你已经在运行：`python -m http.server 8000`

#### 在手机上访问
1. **获取电脑 IP 地址**
   - Windows：打开命令提示符，输入 `ipconfig`
   - 找到 "IPv4 地址"，例如：`192.168.1.100`

2. **确保电脑和手机在同一 WiFi**
   - 连接到相同的 WiFi 网络

3. **在手机浏览器访问**
   - 打开手机浏览器
   - 输入：`http://192.168.1.100:8000/graduation.html`
   - 将 `192.168.1.100` 替换为你的实际 IP 地址

#### 注意事项
- 电脑必须保持开机和运行服务器
- 只能在同一 WiFi 下访问
- 关闭电脑或停止服务器后无法访问

## ☁️ 方式三：使用云存储服务

### 选项 A：使用 Gitee Pages（国内访问快）

#### 步骤 1：注册 Gitee
1. 访问 https://gitee.com
2. 注册免费账号

#### 步骤 2：创建仓库
1. 点击右上角 "+" → "新建仓库"
2. 仓库名称：`graduation-board`
3. 选择 "公开"
4. 点击 "创建"

#### 步骤 3：上传文件
1. 在仓库页面点击 "上传文件"
2. 上传所有项目文件
3. 提交更改

#### 步骤 4：开启 Gitee Pages
1. 在仓库页面找到 "服务" → "Gitee Pages"
2. 点击 "启动"
3. 等待部署完成

#### 步骤 5：访问
- 访问地址：`https://你的用户名.gitee.io/graduation-board/graduation.html`

## ⚠️ 重要提示

### 数据存储说明
- 数据保存在每个用户的浏览器本地
- 不同设备的数据是独立的
- 清除浏览器数据会丢失留言
- 建议定期截图保存重要留言

### 隐私和安全
- 使用公开托管平台时，任何人都可以访问
- 私密留言只在本地浏览器可见
- 不要在留言中包含敏感个人信息
- 建议修改默认密码

### 性能优化
- 照片建议压缩后上传（小于1MB）
- 避免上传过大的文件
- 定期清理不需要的留言

## 🆘 常见问题

**Q: 手机上看不到样式怎么办？**
A: 检查 CSS 文件路径是否正确，确保所有文件都已上传。

**Q: 多个同学可以同时使用吗？**
A: 可以，但每个人的数据是独立的，保存在各自的浏览器中。

**Q: 如何备份留言数据？**
A: 建议使用截图或浏览器的打印功能保存重要留言。

**Q: 可以修改界面样式吗？**
A: 可以，修改 `css/graduation.css` 文件即可。

**Q: 忘记密码怎么办？**
A: 默认密码与用户名相同，清除浏览器数据可重置。
